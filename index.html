eBPF жҳҜеөҢе…ҘеңЁ Linux еҶ…ж ёдёӯзҡ„иҷҡжӢҹжңәгҖӮе®ғе…Ғи®ёе°Ҷе°ҸзЁӢеәҸеҠ иҪҪеҲ°еҶ…ж ёдёӯпјҢе№¶йҷ„еҠ еҲ°й’©еӯҗдёҠпјҢеҪ“жҹҗдәӣдәӢд»¶еҸ‘з”ҹж—¶дјҡи§ҰеҸ‘иҝҷдәӣй’©еӯҗгҖӮиҝҷе…Ғи®ёпјҲжңүж—¶еӨ§йҮҸпјүе®ҡеҲ¶еҶ…ж ёзҡ„иЎҢдёәгҖӮиҷҪз„¶ eBPF иҷҡжӢҹжңәеҜ№дәҺжҜҸз§Қзұ»еһӢзҡ„й’©еӯҗйғҪжҳҜзӣёеҗҢзҡ„пјҢдҪҶй’©еӯҗзҡ„еҠҹиғҪеҚҙеӨ§дёҚзӣёеҗҢгҖӮз”ұдәҺе°ҶзЁӢеәҸеҠ иҪҪеҲ°еҶ…ж ёдёӯеҸҜиғҪеҫҲеҚұйҷ©пјӣеҶ…ж ёйҖҡиҝҮйқһеёёдёҘж јзҡ„йқҷжҖҒйӘҢиҜҒеҷЁиҝҗиЎҢжүҖжңүзЁӢеәҸпјӣйӘҢиҜҒзЁӢеәҸеҜ№зЁӢеәҸиҝӣиЎҢжІҷз®ұеӨ„зҗҶпјҢзЎ®дҝқе®ғеҸӘиғҪи®ҝй—®е…Ғи®ёзҡ„еҶ…еӯҳйғЁеҲҶпјҢе№¶зЎ®дҝқе®ғеҝ…йЎ»еҝ«йҖҹз»ҲжӯўгҖӮ

eBPF иғҪеҒҡд»Җд№Ҳ

eBPF зЁӢеәҸзҡ„зұ»еһӢ

жңүеҮ зұ»й’©еӯҗеҸҜд»ҘеңЁеҶ…ж ёдёӯйҷ„еҠ  eBPF зЁӢеәҸгҖӮeBPF зЁӢеәҸзҡ„еҠҹиғҪеҫҲеӨ§зЁӢеәҰдёҠеҸ–еҶідәҺе®ғжүҖйҷ„еҠ зҡ„й’©еӯҗпјҡ

  • и·ҹиёӘзЁӢеәҸеҸҜд»Ҙйҷ„еҠ еҲ°еҶ…ж ёдёӯеҫҲеӨ§дёҖйғЁеҲҶеҠҹиғҪдёҠгҖӮи·ҹиёӘзЁӢеәҸеҜ№дәҺж”¶йӣҶз»ҹи®ЎдҝЎжҒҜе’Ңж·ұе…Ҙи°ғиҜ•еҶ…ж ёеҫҲжңүз”ЁгҖӮ еӨ§еӨҡж•°и·ҹиёӘжҢӮй’©еҸӘе…Ғи®ёеҜ№еҮҪж•°жӯЈеңЁеӨ„зҗҶзҡ„ж•°жҚ®иҝӣиЎҢеҸӘиҜ»и®ҝй—®пјҢдҪҶд№ҹжңүдёҖдәӣе…Ғи®ёдҝ®ж”№ж•°жҚ®гҖӮдҫӢеҰӮпјҢжүҫеҮәеҶ…ж ёж„ҸеӨ–дёўејғж•°жҚ®еҢ…зҡ„еҺҹеӣ гҖӮ

  • жөҒйҮҸжҺ§еҲ¶(tc) зЁӢеәҸеҸҜд»ҘеңЁе…ҘеҸЈе’ҢеҮәеҸЈеӨ„йҷ„еҠ еҲ°з»ҷе®ҡзҡ„зҪ‘з»ңи®ҫеӨҮгҖӮеҶ…ж ёдёәжҜҸдёӘж•°жҚ®еҢ…жү§иЎҢдёҖж¬ЎзЁӢеәҸгҖӮз”ұдәҺй’©еӯҗз”ЁдәҺж•°жҚ®еҢ…еӨ„зҗҶпјҢеҶ…ж ёе…Ғи®ёзЁӢеәҸдҝ®ж”№жҲ–жү©еұ•ж•°жҚ®еҢ…пјҢдёўејғж•°жҚ®еҢ…пјҢе°Ҷе…¶ж Үи®°дёәжҺ’йҳҹпјҢжҲ–е°Ҷж•°жҚ®еҢ…йҮҚе®ҡеҗ‘еҲ°еҸҰдёҖдёӘжҺҘеҸЈгҖӮCalico зҡ„ eBPF ж•°жҚ®е№ійқўе°ұжҳҜеҹәдәҺиҝҷз§Қзұ»еһӢзҡ„й’©еӯҗпјӣCalico дҪҝз”Ё tc зЁӢеәҸеҜ№ Kubernetes жңҚеҠЎиҝӣиЎҢиҙҹиҪҪеқҮиЎЎпјҢе®һж–ҪзҪ‘з»ңзӯ–з•ҘпјҢе№¶дёәе·Іе»әз«ӢиҝһжҺҘзҡ„жөҒйҮҸеҲӣе»әеҝ«йҖҹи·Ҝеҫ„гҖӮ

  • XDPжҲ–вҖңeXpress ж•°жҚ®и·Ҝеҫ„вҖқе®һйҷ…дёҠжҳҜ eBPF жҢӮй’©зҡ„еҗҚз§°гҖӮжҜҸдёӘзҪ‘з»ңи®ҫеӨҮйғҪжңүдёҖдёӘ XDP е…ҘеҸЈжҢӮй’©пјҢеңЁеҶ…ж ёдёәж•°жҚ®еҢ…еҲҶй…ҚеҘ—жҺҘеӯ—зј“еҶІеҢәд№ӢеүҚпјҢжҜҸдёӘдј е…Ҙж•°жҚ®еҢ…йғҪдјҡи§ҰеҸ‘дёҖж¬ЎиҜҘжҢӮй’©гҖӮXDP еҸҜд»Ҙдёә DoS дҝқжҠӨпјҲеңЁ Calico зҡ„ж ҮеҮҶ Linux ж•°жҚ®е№ійқўдёӯж”ҜжҢҒпјүе’Ңе…ҘеҸЈиҙҹиҪҪе№іиЎЎпјҲеңЁ facebook зҡ„ Katran дёӯдҪҝз”Ёпјүзӯүз”ЁдҫӢжҸҗдҫӣеҮәиүІзҡ„жҖ§иғҪгҖӮXDP зҡ„зјәзӮ№жҳҜе®ғйңҖиҰҒзҪ‘з»ңи®ҫеӨҮй©ұеҠЁзЁӢеәҸж”ҜжҢҒжүҚиғҪиҺ·еҫ—иүҜеҘҪзҡ„жҖ§иғҪгҖӮXDP жң¬иә«дёҚи¶ід»Ҙе®һзҺ° Kubernetes pod зҪ‘з»ңжүҖйңҖзҡ„жүҖжңүйҖ»иҫ‘пјҢдҪҶ XDP е’ҢжөҒйҮҸжҺ§еҲ¶жҢӮй’©зҡ„з»„еҗҲж•ҲжһңеҫҲеҘҪгҖӮ

  • еҘ—жҺҘеӯ—зЁӢеәҸжҢӮй’©еҲ°еҘ—жҺҘеӯ—дёҠзҡ„еҗ„з§Қж“ҚдҪңпјҢдҫӢеҰӮпјҢе…Ғи®ё eBPF зЁӢеәҸжӣҙж”№ж–°еҲӣе»әзҡ„еҘ—жҺҘеӯ—зҡ„зӣ®ж Ү IPпјҢжҲ–ејәеҲ¶еҘ—жҺҘеӯ—з»‘е®ҡеҲ°вҖңжӯЈзЎ®зҡ„вҖқжәҗ IP ең°еқҖгҖӮCalico дҪҝз”ЁжӯӨзұ»зЁӢеәҸеҜ№ Kubernetes жңҚеҠЎиҝӣиЎҢиҝһжҺҘж—¶иҙҹиҪҪеқҮиЎЎпјӣиҝҷеҮҸе°‘дәҶејҖй”ҖпјҢеӣ дёәж•°жҚ®еҢ…еңЁеӨ„зҗҶи·Ҝеҫ„дёҠжІЎжңүеҒҡDNATгҖӮ

  • жңүеҗ„з§ҚдёҺе®үе…Ёзӣёе…ізҡ„й’©еӯҗе…Ғи®ёд»Ҙеҗ„з§Қж–№ејҸзӣ‘з®ЎзЁӢеәҸиЎҢдёәгҖӮдҫӢеҰӮпјҢseccompжҢӮй’©е…Ғи®ёд»Ҙз»ҶзІ’еәҰзҡ„ж–№ејҸзӣ‘з®Ўзі»з»ҹи°ғз”ЁгҖӮ

еҶ…ж ёйҖҡиҝҮhelper functionsе…¬ејҖжҜҸдёӘй’©еӯҗзҡ„еҠҹиғҪгҖӮдҫӢеҰӮпјҢtc й’©еӯҗжңүдёҖдёӘ helper function жқҘи°ғж•ҙж•°жҚ®еҢ…зҡ„еӨ§е°ҸпјҢдҪҶиҜҘеё®еҠ©еҮҪж•°еңЁи·ҹиёӘй’©еӯҗдёӯдёҚеҸҜз”ЁгҖӮдҪҝз”Ё eBPF зҡ„жҢ‘жҲҳд№ӢдёҖжҳҜдёҚеҗҢзҡ„еҶ…ж ёзүҲжң¬ж”ҜжҢҒдёҚеҗҢзҡ„еҠ©жүӢпјҢиҖҢзјәе°‘еҠ©жүӢеҸҜиғҪдјҡеҜјиҮҙж— жі•е®һзҺ°зү№е®ҡзҡ„еҠҹиғҪгҖӮ

Calico зҡ„ eBPF ж•°жҚ®е№ійқў

Calico зҡ„ eBPF ж•°жҚ®е№ійқўжҳҜж ҮеҮҶ Linux ж•°жҚ®е№ійқўпјҲеҹәдәҺ iptablesпјүзҡ„жӣҝд»Јж–№жЎҲгҖӮж ҮеҮҶж•°жҚ®е№ійқўдҫ§йҮҚдәҺйҖҡиҝҮдёҺ kube-proxy е’Ң iptables и§„еҲҷиҝӣиЎҢдәӨдә’жқҘе®һзҺ°е…је®№жҖ§пјҢиҖҢ eBPF ж•°жҚ®е№ійқўеҲҷдҫ§йҮҚдәҺжҖ§иғҪгҖҒе»¶иҝҹе’Ңж”№е–„з”ЁжҲ·дҪ“йӘҢпјҢе№¶жҸҗдҫӣж ҮеҮҶж•°жҚ®е№ійқўдёӯж— жі•е®һзҺ°зҡ„еҠҹиғҪгҖӮдҪңдёәе…¶дёӯзҡ„дёҖйғЁеҲҶпјҢeBPF ж•°жҚ®е№ійқўе°Ҷ kube-proxy жӣҝжҚўдёә eBPF е®һзҺ°гҖӮдё»иҰҒзҡ„вҖңз”ЁжҲ·дҪ“йӘҢвҖқеҠҹиғҪжҳҜеңЁжөҒйҮҸеҲ°иҫҫ NodePort ж—¶дҝқз•ҷжқҘиҮӘйӣҶзҫӨеӨ–йғЁзҡ„жөҒйҮҸжәҗ IPпјӣиҝҷдҪҝжңҚеҠЎеҷЁз«Ҝж—Ҙеҝ—е’ҢзҪ‘з»ңзӯ–з•ҘеңЁиҜҘи·Ҝеҫ„дёҠжӣҙеҠ жңүз”ЁгҖӮ

ж–°зҡ„ж•°жҚ®е№ійқўдёҺ Calico зҡ„ж ҮеҮҶLinuxзҪ‘з»ңж•°жҚ®е№ійқўзӣёжҜ”

  • е®ғеҸҜд»Ҙжү©еұ•еҲ°жӣҙй«ҳзҡ„еҗһеҗҗйҮҸгҖӮ
  • е®ғжҜҸ GBit дҪҝз”Ёжӣҙе°‘зҡ„ CPUгҖӮ

  • е®ғеҺҹз”ҹж”ҜжҢҒ Kubernetes жңҚеҠЎпјҲдёҚйңҖиҰҒ kube-proxyпјүпјҡ

    • еҮҸе°‘ж•°жҚ®еҢ…еҲ°жңҚеҠЎзҡ„з¬¬дёҖдёӘж•°жҚ®еҢ…е»¶иҝҹгҖӮ
    • е°ҶеӨ–йғЁе®ўжҲ·з«Ҝжәҗ IP ең°еқҖдёҖзӣҙдҝқз•ҷеҲ° podгҖӮ
    • ж”ҜжҢҒ DSRпјҲDirect Server ReturnпјүпјҢе®һзҺ°жӣҙй«ҳж•Ҳзҡ„жңҚеҠЎи·Ҝз”ұгҖӮ
    • дҪҝз”ЁжҜ” kube-proxy жӣҙе°‘зҡ„ CPU жқҘдҝқжҢҒж•°жҚ®е№ійқўеҗҢжӯҘгҖӮ

иҰҒдәҶи§ЈжӣҙеӨҡзҡ„жҖ§иғҪжҢҮж ҮпјҢиҜ·еҸӮйҳ…еҚҡе®ў д»Ӣз»Қ Calico eBPF ж•°жҚ®е№ійқў

йҷҗеҲ¶

eBPF жЁЎејҸзӣ®еүҚзӣёеҜ№дәҺж ҮеҮҶ Linux з®ЎйҒ“жЁЎејҸжңүдёҖдәӣйҷҗеҲ¶пјҡ

  • eBPF жЁЎејҸд»…ж”ҜжҢҒ x86-64гҖӮпјҲзӣ®еүҚжІЎжңүдёәе…¶д»–е№іеҸ°жһ„е»ә eBPF зЁӢеәҸгҖӮпјү
  • eBPF жЁЎејҸе°ҡдёҚж”ҜжҢҒ IPv6гҖӮ
  • еҗҜз”Ё eBPF жЁЎејҸж—¶пјҢйў„е…ҲеӯҳеңЁзҡ„иҝһжҺҘз»§з»ӯдҪҝз”Ёйқһ BPF ж•°жҚ®и·Ҝеҫ„пјӣиҝҷж ·зҡ„иҝһжҺҘдёҚеә”иҜҘиў«жү“ж–ӯпјҢдҪҶе®ғд»¬е№¶дёҚиғҪд»Һ eBPF жЁЎејҸзҡ„дјҳеҠҝдёӯеҸ—зӣҠгҖӮ
  • зҰҒз”Ё eBPF жЁЎејҸе…·жңүз ҙеқҸжҖ§пјӣйҖҡиҝҮ eBPF ж•°жҚ®е№ійқўеӨ„зҗҶзҡ„иҝһжҺҘеҸҜиғҪдјҡдёӯж–ӯпјҢе№¶дё”еҸҜиғҪйңҖиҰҒйҮҚж–°еҗҜеҠЁжңӘжЈҖжөӢе’ҢжҒўеӨҚзҡ„жңҚеҠЎгҖӮ
  • дёҚж”ҜжҢҒж··еҗҲйӣҶзҫӨпјҲеёҰжңүдёҖдәӣ eBPF иҠӮзӮ№е’ҢдёҖдәӣж ҮеҮҶж•°жҚ®е№ійқўиҠӮзӮ№пјүгҖӮпјҲеңЁиҝҷж ·зҡ„йӣҶзҫӨдёӯпјҢд»Һ eBPF иҠӮзӮ№еҲ°йқһ eBPF иҠӮзӮ№зҡ„ NodePort жөҒйҮҸе°Ҷиў«дёўејғгҖӮпјүиҝҷеҢ…жӢ¬е…·жңү Windows иҠӮзӮ№зҡ„йӣҶзҫӨгҖӮ
  • eBPF жЁЎејҸдёҚж”ҜжҢҒжө®еҠЁ IPгҖӮ

Calico зҡ„ eBPF ж•°жҚ®е№ійқў

Calico зҡ„ eBPF ж•°жҚ®е№ійқўжҳҜж ҮеҮҶ Linux ж•°жҚ®е№ійқўпјҲеҹәдәҺ iptablesпјүзҡ„жӣҝд»Јж–№жЎҲгҖӮж ҮеҮҶж•°жҚ®е№ійқўдҫ§йҮҚдәҺйҖҡиҝҮдёҺ kube-proxy е’Ң iptables и§„еҲҷиҝӣиЎҢдәӨдә’жқҘе®һзҺ°е…је®№жҖ§пјҢиҖҢ eBPF ж•°жҚ®е№ійқўеҲҷдҫ§йҮҚдәҺжҖ§иғҪгҖҒе»¶иҝҹе’Ңж”№е–„з”ЁжҲ·дҪ“йӘҢпјҢе№¶жҸҗдҫӣж ҮеҮҶж•°жҚ®е№ійқўдёӯж— жі•е®һзҺ°зҡ„еҠҹиғҪгҖӮдҪңдёәе…¶дёӯзҡ„дёҖйғЁеҲҶпјҢeBPF ж•°жҚ®е№ійқўе°Ҷ kube-proxy жӣҝжҚўдёә eBPF е®һзҺ°гҖӮдё»иҰҒзҡ„вҖңз”ЁжҲ·дҪ“йӘҢвҖқеҠҹиғҪжҳҜеңЁжөҒйҮҸеҲ°иҫҫ NodePort ж—¶дҝқз•ҷжқҘиҮӘйӣҶзҫӨеӨ–йғЁзҡ„жөҒйҮҸжәҗ IPпјӣиҝҷдҪҝжңҚеҠЎеҷЁз«Ҝж—Ҙеҝ—е’ҢзҪ‘з»ңзӯ–з•ҘеңЁиҜҘи·Ҝеҫ„дёҠжӣҙеҠ жңүз”ЁгҖӮ
