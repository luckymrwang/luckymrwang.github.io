发表于 2017-09-11

折腾了很久后最终发现是nvm的原因

所以列出我的解决办法

  • 安装 brew install nvm
  • 创建文件目录 mkdir ~/.nvm
  • 设置环境变量，将如下信息添加到~/.bash_profile中
  • source ~/.bash_profile
  • 用nvm安装Node.js nvm install stable
  • npm install -g hexo-cli
