Html与MySQL空格的转换

  • MySQL
  • PHP

在HTML的文本文件域 <textarea> 中保存好换行格式的文本文字后，存储到MySQL里，之后再取出显示到网页上时，所有文字都在一行上失去了之前的换行格式。

原因：
文本域中的换行符是\n,存储到MySQL的也是该符号，但在html中需要换行的符号必须是<br \>,所以就造成了两边格式的不一致。

解决办法：
从库中取出的数据输出到html前用PHP自带函数nl2br() 在字符串中的每个新行 (\n) 之前插入 HTML 换行符 (
)处理一下
